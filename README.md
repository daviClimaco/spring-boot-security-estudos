# Biblioteca Virtual

Projeto desenvolvido em Java utilizando Spring Boot para simular uma biblioteca virtual.

O sistema permitirá o cadastro e gerenciamento de livros, além de autenticação e autorização de usuários utilizando Spring Security e JWT.

## Tecnologias

* Java
* Spring Boot
* Spring Security
* JWT
* Maven
* Banco de Dados Relacional
